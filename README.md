# 🧠 JavaScript Quiz Game

A simple **interactive quiz game built using Vanilla JavaScript, HTML, and CSS**.
The application presents multiple-choice questions, allows users to select answers, and displays the final score at the end.

This project demonstrates **DOM manipulation, event handling, and basic game logic in JavaScript**.

---

## 🎮 How to Play

1. Click **Start Quiz**
2. Read the question carefully
3. Select the correct option
4. Move to the next question
5. At the end, your **final score** will be displayed

---

## ✨ Features

* ❓ Multiple-choice questions
* ⚡ Instant answer selection
* 📊 Score calculation
* 🔄 Restart quiz option
* 🎨 Clean and simple UI
